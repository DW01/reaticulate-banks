# Kevin's Reaticulate Banks

This repository contains, organised by vendor, series (if applicable) and library, a collection of Reaticulate bank files for primarily my own personal use, that I've decided to make available on GitHub for free use by anyone, whether in commercial projects or not. I made these to fit my workflow, but as Reaticulate banks are essentially fancy text files, you can load them up in an editor of your choice (I personally use [Zed](https://zed.dev/)) and tweak the parameters to your liking.

You may also notice that I use a lot of bank duplication. This is a matter of preference; while it'd certainly be more efficient to simply name the main CSS bank "Strings" and be done with it instead of cloning the First Violins bank three times, I like to know which track is assigned which instrument / section at a glance in the Reaticulate window. Yes, the Second Violins, Violas and Celli banks are identical to the First Violins bank, that's intentional.

## Individual notes
- **Cinematic Studio Series:** These are keyswitch-based banks. Make sure your delay solution (Cinematic Delay Compensator, CSS Delay Helper, VDC, Foresight...) both supports and is set to KEYSWITCH mode.
- **True Strike:** There's a lot of blank banks here, purely for organisational purposes. Feel free to remove them in your forks.
- **Kontakt Factory Library:** This one only contains the KFL v1 Vienna harp for now. I might make this into a full library bank set at some point, but that's a loooot of work.
- **Kontakt Factory Library 2:** The Strezov Sampling vowels are untested. They should work, but due to the dual KS system, the program numbers are a bit odd in places.
